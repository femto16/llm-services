# LLM-Services

- LMStudio
    - ローカルLLMサーバ
- Mattermost
    - 通知、チャット、n8nなどのトリガー実行用プラットフォーム
- n8n
    - 各種自動化、LLMエージェントなどのワークフローの作成
    - ワークフローのトリガー実行、定期実行
