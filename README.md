# mattermost-n8n-bot

- WSLのインストール
- docker環境の作成

# n8nのセットアップ
- setup_n8n.batを実行
- launch_n8n.batを実行
- http://localhost:5678 を開く
- アカウントを作成する

# Mattermostのセットアップ
- setup_mattermost.batを実行
- launch_mattermost.batを実行
- http://localhost:8065 を開く
- アカウントを作成する
- Organizationを作成する
- 「What tools do you use?」はskip
- 左上のメニュー System Console → CORS → Enable cross-origin requests from: に「*」を入力してSave
